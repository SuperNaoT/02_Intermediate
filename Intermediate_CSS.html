　■　２つの要素を重ねる。【position: relative(absolute);】

　　　　【参照：https://prog-8.com/html/study/2/13#/37】
　　　　　　※　HTML中の各要素は、表示上重なる事はない。
　　　　　　　　しかしながらその表示位置を【position】（CSS）にて指定すると、表示位置を指定出来表示を重ねる事が出来る！
　　　　　　　　1) position: relative;　・・・この指定を入れた親要素の左上が原点と成る（指定無しは、画面左上が原点）。
　　　　　　　　2) position: absolute;　・・・親要素の原点からの絶対位置関係、そこからのピクセル位置で指定する。
　　　　　《HTML記載例》
　　　　　　　　　　<div class="parent-pkg">
　　　　　　　　　　　　<img class="child-img" src="https://prog-8.com/images/html/advanced/html.png">
　　　　　　　　　　　　<p class="child-text">HTML & CSS</p>
　　　　　　　　　　</div>
　　　　　《CSS記載例》
　　　　　　　　　　.parent-pkg {
　　　　　　　　　　　　position: relative;　・・・【親・原点指定】
　　　　　　　　　　　　width: 300px;
　　　　　　　　　　　　height: 300px;
　　　　　　　　　　　　text-align: center;
　　　　　　　　　　　　border-style: dashed;
　　　　　　　　　　　　background-color: yellowgreen;
　　　　　　　　　　}
　　　　　　　　　　.child-img {
　　　　　　　　　　　　position: absolute;　・・・【子・原点からの距離指定明示】
　　　　　　　　　　　　top: 50px;
　　　　　　　　　　　　left: 50px;
　　　　　　　　　　}
　　　　　　　　　　.child-text {
　　　　　　　　　　　　position: absolute;　・・・【子・原点からの距離指定明示】
　　　　　　　　　　　　top: 143px;
　　　　　　　　　　　　left: 100px;
　　　　　　　　　　}
　　　　【実行結果・・・】

　■　相対的な（親に対する）大きさ指定の際での、レイアウト崩れを避けるには・・・【box-sizing: border-box;】

　　　　【参照：https://prog-8.com/html/study/3/4#/8】
　　　　　　※　相対的な大きさ指定を「％」で行った場合、そのサイズがきっちりしたサイズであると、
　　　　　　　　「padding」等余白を指定すると「１００％を超え」てしまい、レイアウトが崩れる。
　　　　　　　　「box-sizing: border-box;」を用いる事にて、レイアウト崩れを避ける事が出来る。
 　　　　　《HTML記載例》
　　　　　　　　　　<div class="container">
　　　　　　　　　　　　<div class="item">
　　　　　　　　　　　　　　<div class="item-box green"></div>
　　　　　　　　　　　　</div>
　　　　　　　　　　　　<div class="item">
　　　　　　　　　　　　　　<div class="item-box yellow"></div>
　　　　　　　　　　　　</div>
　　　　　　　　　　　　<div class="item">
　　　　　　　　　　　　　　<div class="item-box red"></div>
　　　　　　　　　　　　</div>
　　　　　　　　　　　　<div class="item">
　　　　　　　　　　　　　　<div class="item-box blue"></div>
　　　　　　　　　　　　</div>
　　　　　　　　　　</div>
　　　　　《CSS記載例》
　　　　　　　　　　.container { margin: 20px; }
　　　　　　　　　　.item {
　　　　　　　　　　　　width: 25%;
　　　　　　　　　　　　float: left;
　　　　　　　　　　}
　　　　　　　　　　.item-box {
　　　　　　　　　　　　height: 100px;
　　　　　　　　　　　　margin-bottom: 20px;
　　　　　　　　　　}
　　　　　　　　　　.green { background-color: #11D8B3; }
　　　　　　　　　　.yellow { background-color: #FFD466; }
　　　　　　　　　　.red { background-color: #ED456D; }
　　　　　　　　　　.blue { background-color: #3D7DD8; }
　　　　　　　　　　/* タブレット向けレイアウト */
　　　　　　　　　　@media (max-width: 1000px) {
　　　　　　　　　　　　.item { width: 50%; }
　　　　　　　　　　}
　　　　　　　　　　/* スマートフォン向けレイアウト */
　　　　　　　　　　@media (max-width: 670px) {
　　　　　　　　　　　　.item { width: 100%; }
　　　　　　　　　　}
　　　　【実行結果・・・】
