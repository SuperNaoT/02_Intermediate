　■　文字列の一部分に特例スタイルを指定する為のタグ！【<span> </span>】

　　　　【参照：https://prog-8.com/html/study/1/18#/59】
　　　　　※　表示文字列の一部、例えば「Progate」だけを赤文字に。

　　　　　《HTML記載例》
　　　　　　　　<div class="copy-container">　私は「<span>Progate</span>」にてHTML/CSSを勉強しています。　</div>
　　　　　《CSS記載例》
　　　　　　　　.copy-container span {　color: #ff0000;　}
　　　　【実行結果・・・】
　　　　　　　　私は「Progate」にてHTML/CSSを勉強しています。

　■　インライン要素・インラインブロック要素・ブロック要素！【違いについて】

　　　　【参照：https://prog-8.com/html/study/2/5#/11】
　　　　　※《HTML インライン要素一覧》
　　　　　※《HTML ブロック要素　一覧》
　　　　　
　　　　※　インライン・ブロック要素に、display (CSS) 属性を指定する事により、
　　　　　　それぞれ要素を「インライン・インラインブロック・ブロック要素」に変更できる。
　　　　　　・・・「width/height, margin/padding」を指定出る様に成る。
　　　　　《CSS記載例》
　　　　　　　　.inlineElement {
　　　　　　　　　　display: inline;　　　　 ・・・インライン要素扱いと成る。
　　　　　　　　　　display: inline-block;　・・・インラインブロック要素扱いとなる。
　　　　　　　　　　display: block;　　　　 ・・・ブロック要素扱いとなる。
　　　　　　　　}
